# 🐍 Juego de la Serpiente
Proyecto desarrollado en *Godot Engine 4.2.2* para la materia de Desarrollo de Videojuegos.

## 📋 Descripción
Juego clásico con 5 niveles de dificultad progresiva, laberintos, sonidos y sistema de puntuación.

## 🎮 Controles
- Flechas: Mover
- Espacio: Iniciar / Reiniciar

## 📦 Características
✅ 5 Niveles diseñados
✅ Velocidad aumenta por nivel
✅ Mensajes: "Puntos", "Nivel", "Chocaste con pared"
✅ Sonidos y efectos
✅ Exportado para: Windows, Linux, Android

## 📂 Estructura
- /imagenes/ → Recursos gráficos
- /sonidos/ → Efectos de audio
- project.godot → Archivo principal del proyecto
- snake.gd → Código completo
